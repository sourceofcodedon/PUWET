PampangNav Web App - Setup Guide

Prerequisites
- Computer with internet connection
- Modern web browser (Chrome, Firefox, Safari, Edge)
- VS Code (or any IDE with Live Server extension)

Quick Setup Steps
1. Install VS Code Extension
- Open VS Code
- Go to Extensions (Ctrl+Shift+X)
- Search for "Live Server" by Ritwick Dey
- Install it

2. Open Project
- Open the PampangNavWeb folder in VS Code
- Right-click on login.html in the file explorer
- Select "Open with Live Server"

3. That's it! The app will open in your browser
First Time Setup
- The app will open to the login page
- Click "SIGN UP" to create your first admin account
- After signup, login with your credentials
- You'll be redirected to the dashboard
